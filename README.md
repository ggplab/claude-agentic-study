## 운영 방식 (첫 모임에서 확정 — 둘 다 열어둠)

이 책은 챕터마다 "무엇을 해야 하는가"가 있습니다. 진행 방식은 두 가지 다 가능하고, 첫 모임에서 합의합니다.

- **A. 리드 발표형** — 챕터마다 리드를 정해 핵심을 발표
- **B. 각자 빌드 공유형** — 각자 그 챕터에서 필요한 걸 하나 만들어 와 결과물·시행착오 공유

> 한 챕터 안에서 둘을 섞어도 됩니다. (예: 리드가 개요 → 각자 만든 것 공유)

**기록 단위**: 챕터당 `chapters/chNN/README.md` 한 장에 다 같이 모읍니다. (개인 결과물은 링크로)
