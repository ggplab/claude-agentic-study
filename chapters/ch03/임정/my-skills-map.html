② 대표 프론트매터 — 최소형에서 안전·권한·모델까지
commit최소 형태 — 인자 없음, 자동호출만
name: commit
description: Stage and commit … Use when the user asks to commit.
order-session레포에 묶인 인자형 (프로젝트 스킬)
name: order-session
description: 단원 리더가 발표 순서를 조정한다 …
argument-hint: <챕터번호 (예 3)>
study-chapter안전장치 — 커밋·푸시하므로 자동호출 차단
name: study-chapter
description: … 정리본만 커밋·푸시 …
argument-hint: <챕터번호> [이름]
disable-model-invocation: true   # /로만 호출
review-inquiry권한을 스킬 단위로 좁힘 (Notion 도구만)
name: review-inquiry
description: 강의/컨설팅 문의 접수 → 판정 → Notion 기록
allowed-tools: Read Write Bash
  mcp__…Notion__notion-search
  mcp__…Notion__notion-create-pages
yc-office-hours판단 무거운 스킬은 모델 격상
name: yc-office-hours
description: 사업 결정을 강제 질문으로 검증 …
argument-hint: [검토할 아이디어나 결정]
model: opus
argument-hint인자를 받는 스킬 — / 입력 시 자동완성 힌트로 뜬다
disable-model-invocation비가역 사이드이펙트의 안전장치. 내 시스템에선 전송·생성·커밋하는 스킬(discord-channel · meeting-minutes · study-chapter · chapter-deck)에만 붙어 있다 — 책 표 3-1이 빠뜨린 필드
allowed-tools도구 사전승인 → 권한 프롬프트 절약. 특정 도구만 적으면 그 스킬의 권한 범위를 좁히는 효과
model그 스킬이 도는 동안 모델 격상/격하 (이번 턴 한정)
정직 표기. “대표”는 워크플로에서의 역할 기준으로 골랐다(측정된 호출 횟수가 아님). 실호출 집계는 infra-inventory 스킬이 트랜스크립트로 따로 낸다. 프론트매터는 실제 SKILL.md에서 발췌·요약. · ggplab brand-and-light (warm-paper) 토큰 적용.
